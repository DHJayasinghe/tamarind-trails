# Tamarind Trails

A minimalistic single-page website promoting Sri Lanka tourism for **Tamarind Trails** — your mom's new tour guide business.

## Overview

A static, single-page website inspired by Walker's Tours, featuring:

- **Hero** — Full-screen intro with Sigiriya imagery
- **About** — Company story and value proposition
- **Tours** — Four tour categories (Cultural, Hill Country, Beach, Wildlife)
- **Why Choose Us** — Key differentiators
- **CTA** — Call-to-action with Ella imagery
- **Contact** — Contact info and inquiry form

## Design

- Minimalistic layout with warm earth tones
- Cormorant Garamond + Outfit typography
- Unsplash placeholder images (replace with custom photos when ready)
- Responsive for desktop, tablet, and mobile
- Simple navigation: Home, About, Tours, Contact

## Getting Started

1. Open `index.html` in a browser, or
2. Use a local server (e.g. `npx serve .` or VS Code Live Server) for best results

## Customization

- **Images**: Replace Unsplash URLs in `index.html` with your own image paths
- **Contact details**: Update email and phone in the Contact section
- **Content**: Edit copy directly in `index.html`
- **Colors**: Adjust CSS variables in `styles.css` (`:root` section)

## Tech Stack

- Vanilla HTML5, CSS3, JavaScript
- Google Fonts (Cormorant Garamond, Outfit)
- Unsplash images (placeholders)
